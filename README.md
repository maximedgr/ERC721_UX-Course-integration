# ERC721_UX-Course-integration

This repo is a solution to the [following school project](https://github.com/AymericNoel/erc721-ux)

First clone this repo or check the vercel live integration.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### Infos

The purpose of this project is to perform a number of actions via different "fake nft" contracts deployed on the Seploia testnet.

Feel free to navigate accross the website.
